# Omkar Jadhav | Portfolio 2.0 🚀

Welcome to the repository for my second-generation personal portfolio. This project serves as a comprehensive showcase of my skills, projects, and experiences as a Frontend Engineer & Full Stack Builder, wrapped in a modern, highly interactive, and performant web application.

## ✨ Key Features

- **Modern UI/UX**: Designed with a sleek, dark-themed interface, featuring modern glassmorphism elements, custom glows, and a curated orange (`#f96d00`) accent theme.
- **Dynamic Projects Showcase**: Elegant, interactive project cards built with robust Tailwind CSS grids showcasing my latest works including NIVRA-Health and PAY KARO apps.
- **Comprehensive Skills Section**: A visually striking masonry layout displaying my expertise across Frontend, Backend, Databases, DevOps, Tools, Soft Skills, and Leadership.
- **Smooth Animations**: Powered by Framer Motion, the portfolio employs buttery-smooth scroll-linked animations, parallax effects, and seamless element transitions to engage visitors.
- **Fully Responsive**: Carefully tuned CSS breakpoints ensure a flawless viewing experience seamlessly scaling from ultra-wide desktops down to mobile devices.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite (powered by Rolldown for hyper-fast builds)
- **Styling**: Tailwind CSS v4 & customized vanilla CSS variables
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/omkar30rj-design/portfolio2.0.git
   ```
2. **Navigate to the project directory**
   ```bash
   cd portfolio2.0
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Start the development server**
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173` to view the application.
